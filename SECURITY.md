# Security Policy

DNTB is a local Godot game prototype and does not currently operate a hosted service.

## Reporting a vulnerability

If you find a security issue, please avoid posting exploit details in a public issue. Contact a maintainer privately when possible, or open a minimal public issue asking for a private contact path.

Useful details include:

- A short description of the problem.
- Steps to reproduce.
- Affected files or systems.
- Whether the issue exposes local files, credentials, network access, or player data.

## Supported versions

The main branch is the only supported development line at this stage.
